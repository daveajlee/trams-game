Vehicle Information

After clicking on a box from the Control Screen, the Vehicle Information Screen appears (as shown above). The following information is shown:
Timetable Id: Route number and route schedule number which this vehicle is running.
Vehicle Id: The ID of the vehicle running the service.
Location: Where the vehicle currently is.
Destination: The direction (i.e. terminus) which the vehicle is heading towards.
Delay: The number of minutes late that the vehicle is running.
If the vehicle is running late or you want to make some changes to that vehicle, click "Make Contact" to view the Make Contact screen and refer to the "Make Contact with Vehicle" help topic. If you don't want to make contact, click "Close Window" to return to the control screen.
